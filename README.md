# 🛒 Sales & Retail Analytics — End-to-End Portfolio Project

## Overview
A comprehensive data science project covering the full analytics pipeline
on the Global Superstore dataset (~10,000 orders, 4 years of sales data).

## 📊 Tools & Technologies
| Layer | Tools |
|---|---|
| Languages | Python, R, SQL |
| ML Models | XGBoost, LightGBM, Prophet (forecasting) |
| BI Dashboards | Power BI, Looker Studio, R Shiny, Streamlit |
| Statistical Analysis | SPSS, statsmodels |
| Database | SQLite (portable), PostgreSQL (optional) |

## 🔬 Project Modules
1. **Data Engineering** — Cleaning, feature engineering, SQLite database
2. **Exploratory Data Analysis** — Python (Pandas, Seaborn, Plotly)
3. **Statistical Analysis** — SPSS + Python statsmodels
4. **Machine Learning**
   - Sales Forecasting (Prophet + XGBoost)
   - Customer Segmentation (K-Means RFM)
   - Churn Prediction (LightGBM classifier)
   - Product Recommendation signals
5. **Dashboards** — Power BI · Looker Studio · R Shiny · Streamlit
